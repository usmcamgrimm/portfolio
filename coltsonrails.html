This was my very first Rails application, used to learn the fundamentals of Ruby on Rails.

Project Overview

After completing the Pragmatic Studio courses for Ruby and Ruby on Rails, I wanted toi start a project. Being a huge fan of the Indianapolis Colts, I started this app, not only to work on basic skills, but to alsoincorporate something I love. This app was initially hosted on Render; for performance reasons I chose instead to use the Parklife gem, generating a static build hosted on Netlify.

I learned a lot about basic Ruby and Rails skills, and as I grow, this app will grow with me. I'm looking forward to building out this app's functionality more and more as I progress.
